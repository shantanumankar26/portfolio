Inventory Management System

Objective: To design an SQL database for efficient inventory tracking and automate reporting using Python scripts.

Methods:

Created an SQL schema to manage inventory data, including product ID, stock levels, supplier details, and reorder thresholds.

Developed Python scripts to extract, transform, and load (ETL) data from transactional logs into the database, ensuring real-time updates.

Automated report generation for inventory status and low-stock alerts using Python libraries like Pandas and Matplotlib.

Deliverables:

Provided stakeholders with interactive reports and visualizations in Power BI, highlighting inventory trends and potential shortages.

Implemented an alert system for replenishing low-stock items, reducing stockouts by 25%.

Outcome: Enhanced SQL and Python scripting abilities while optimizing inventory management processes for operational efficiency.
